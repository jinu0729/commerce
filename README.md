## 🛒 Commerce project

commerce 프로젝트는 전자상거래 플랫폼을 구축하기 위한 자바 기반의 웹 애플리케이션입니다. 이 프로젝트는 멀티 모듈로 이루어져 있으며, 회원가입, 주문 처리, 예약 상품구매 등 전자상거래의 핵심 기능을 포함하고
있습니다.

<br>

## 🏃‍♂ Run

```shell
git clone https://github.com/jinu0729/commerce.git
cd commerce
./gradlew build
./gradlew bootRun
```

<br>
